### Albumdle

This is the second version of this project, its a guessing game based on Wordle,Heardle etc.

## Data 
For now the only data available is the 500 Best album by the rolling stones' magazine.
The data has been cleaned and augmented via Pandas and several API such as discogs for the thumbnail and spotify for the popular tracks.

## Deployement
The project is deploy with Vercel and available [here](https://albumdle-iota.vercel.app/)

## Features

- One album to guess each day
- Pretty display
- Responsive
- Admin panel to orchestrate
- Hints generate server side
